## 한스타그램
> 안드로이드 SNS앱만들기 프로젝트
> 프로젝트 기간 2022.11.08 ~ 2022.11.28
> 개발 인원 : 3명

### Description
---
한스타그램은 인스타그램을 클론코딩한 프로젝트입니다.  
Firebase를 이용해 사용자의 계정정보 및 피드, 친구관계를 저장 합니다.  
자신의 일상을 공유하고 친구의 게시글에 좋아요를 눌러보세요.  
스마트폰 뿐만아니라 워치에서도 호환이 되게 만들었습니다.
